

A REST API is an interface that lets systems communicate over HTTP using standard methods:

- **GET** — read data
- **POST** — create data
- **PUT/PATCH** — update data
- **DELETE** — remove data

Resources are identified by URLs (e.g., `/users/42`), and responses are typically JSON. Key principles: stateless requests, uniform interface, and resource-based design.
